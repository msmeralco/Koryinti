# Changelog

All notable changes to the Revolt project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Initial project setup with Expo and React Native
- Complete navigation structure (Root, Tabs, Map stack)
- 14 MVP screens covering full user flow
- TypeScript configuration with strict mode
- ESLint and Prettier for code quality
- GitHub Actions CI/CD pipeline
- Environment variable management with `.env`
- Security features (gitignore, secret scanning)
- Comprehensive documentation (README, DEVELOPMENT, CONTRIBUTING)
- VS Code workspace configuration

### Screens Implemented

- Registration & Add Vehicle (Auth flow)
- Map Home with Find Nearby / Plan Trip
- Nearby Stations list
- Station Profile with details
- Reserve Station with time selection
- Confirm Payment
- Plan Trip input
- Trip Route with suggested stops
- Reservation Details management
- Scan QR (simulated for MVP)
- Rating experience
- User Profile
- Reservations history

### Developer Experience

- Auto-formatting on save
- Pre-commit linting checks
- TypeScript IntelliSense
- Hot module reloading
- Clear error messages
- Structured file organization
